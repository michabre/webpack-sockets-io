# Webpack / Sockets.io Example

Example of using webpack and sockets.io for a simple chat.

## Commands

```bash
## NPM ##

# Start Express Server
npm run start

# Run Webpack - convert TS files and build package
npm run webpack

# Run Jest tests
npm test

# Run Jest tests with full coverage report
npm report

```
